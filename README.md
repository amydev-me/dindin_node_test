### Requirements

- NodeJS
- NPX
- MySQL

### Installation

Switch to the project folder
```
cd /project_folder
npm install
```

### Config

Rename .env.example

Set the database connection in .env

```
    port = 3000
    DB_USERNAME= root
    DB_PASSWORD=password
    DB_DATABASE= database
    DB_HOST =127.0.0.1
```
